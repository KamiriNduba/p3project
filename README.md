Banking Application README

Installation and Dependencies:
Ensure you have Python 3.x installed on your system.
Clone or download the repository containing the banking application.
Navigate to the project directory in your terminal or command prompt.
Setting Up:
Install the required dependencies using pip:
Copy code
pip install sqlalchemy
Make sure you have SQLite installed, which is the default database used by SQLAlchemy.
Usage:
Run the banking application script:
Copy code
python bank_app.py
Follow the on-screen instructions to navigate through the application.
Choose options from the main menu to log in, create a new account, view promotions, or quit the program.
If creating a new account, input a username and password as prompted. Ensure the username is at least 5 characters long and the password is at least 8 characters long.
If logging in, enter the username and password associated with your account.
Once logged in, you can deposit money, withdraw money, check your balance, or delete your account from the banking menu.
To exit the program, select the "Quit the Program" option from the main menu.
Notes:
This application uses SQLAlchemy for database management and SQLite as the default database.
Ensure to input valid information when creating an account or logging in to avoid errors.# p3project
